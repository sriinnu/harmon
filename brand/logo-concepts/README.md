# Harmon Logo Concepts

I added seven root-logo explorations in the same editorial hand-drawn direction as the latest main mark.

- `logo-concept-01-circle.svg`: single listener inside a circular music halo
- `logo-concept-02-chorus.svg`: three people holding a shared music sheet
- `logo-concept-03-bridge.svg`: two listeners connected by one flowing cable
- `logo-concept-04-bloom.svg`: seated listener with music blooming upward
- `logo-concept-05-bridge-premium.svg`: calmer, tighter bridge mark with a cleaner silhouette
- `logo-concept-06-bridge-expressive.svg`: warmer, more human bridge scene with more motion
- `logo-concept-07-bridge-editorial.svg`: offset, asymmetrical bridge composition with a sharper editorial feel

All concepts keep the same paper-warm background, black ink line work, and human-centered product feel so you can compare composition rather than style drift.
